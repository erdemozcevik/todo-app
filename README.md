# ToDo App

Modern ve kullanıcı dostu bir yapılacaklar listesi uygulaması.

React, Tailwind CSS ve Local Storage kullanılarak geliştirilmiştir. Kullanıcılar görev ekleyebilir, tamamlayabilir ve silebilir. Veriler tarayıcıda saklandığı için sayfa yenilendiğinde görevler korunur.

## Ekran Görüntüsü

![ToDo App Screenshot](./src/assets/ToDogit.png)

## Özellikler

- Görev ekleme
- Görev tamamlama
- Görev silme
- Tamamlanan görevleri çizili gösterme
- Verilerin Local Storage üzerinde saklanması
- Sayfa yenilendiğinde görevlerin korunması
- Responsive ve sade kullanıcı arayüzü

## Kullanılan Teknolojiler

- React
- Tailwind CSS
- JavaScript (ES6+)
- Local Storage API
- Vite
